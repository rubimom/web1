JAVA SCRIPT

JABA SCRIPT 는 Netscape에서 개발 한 크로스 플랫폼, 객체 지향 스크립팅 언어로 Netscape 브라우저에서 처음 사용되었습니다. 그러나 Netscape는 ECMA (European Computer Manufacturers Association)와 협력하여 핵심 JavaScript를 기반으로 한 표준화 된 국제 프로그래밍 언어를 제공했습니다. ECMA는 정보 및 통신 시스템에 대한 국제 표준 협회입니다. ECMAScript라고하는이 표준화 된 JavaScript 버전은 표준을 지원하는 모든 애플리케이션에서 동일한 방식으로 작동합니다. 회사는 개방형 표준 언어를 사용하여 JavaScript 구현을 개발할 수 있습니다. ECMA 표준의 첫 번째 버전은 ECMA-262 사양에 문서화되었습니다. ECMA-262 표준은 또한 ISO (International Organization for Standardization)에서 ISO-16262로 승인되었습니다.
